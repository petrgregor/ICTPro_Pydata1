# ICTPro kurz PYDATA1

## Jupyter notebook

- **Interaktivní prostředí:** Umožňuje spouštět bloky kódu nezávisle na
sobě.
- **Literate Programming:** Kombinace živého kódu, textu (Markdown),
rovnic a vizualizací.
- **Webové rozhraní:** Běží v prohlížeči, ale kód se vykonává v lokálním
nebo vzdáleném kernelu.
- **Standard v Data Science:** Ideální pro experimentování a
dokumentaci postupu.

### Instalace
```bash
pip install jupyter
```

### Spuštění
Jupyter notebook server: 
```bash
jupyter notebook
```
